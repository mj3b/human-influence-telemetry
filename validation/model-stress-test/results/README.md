# Model stress-test results

No protocol-conforming model stress-test result is currently available.

A complete result set must contain:

- two unchanged model submissions;
- run metadata for each model;
- contamination attestations;
- deterministic or documented comparison output;
- disagreement classification;
- interpretation boundaries and limitations.

The absence of results does not block a software DOI or maintenance release. It does block any claim that the model stress test has been completed.
