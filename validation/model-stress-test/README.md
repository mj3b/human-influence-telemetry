# Model-based HIT Rubric Stress Test

This directory defines an exploratory model-based stress test for Human Influence Telemetry.

It is separate from the locked human inter-rater protocol `HIT-IRP-CIGNA-001`.

## Purpose

The stress test asks whether two fresh language-model sessions, given the same frozen public evidence packet and the same HIT instructions, expose ambiguities in:

- evidence selection;
- evidentiary weight;
- assessment boundary;
- construct interpretation;
- `0` versus `IE`;
- ordinal thresholds;
- Telemetry Integrity.

## Non-claims

This exercise does not establish human inter-rater reliability, satisfy claim H3, advance Maturity Level 2, or replace the two independent human scorers required by the locked protocol.

A model agreement result is not evidence that human reviewers will agree. A model disagreement result may still identify useful rubric weaknesses.
