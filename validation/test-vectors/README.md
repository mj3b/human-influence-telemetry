# Synthetic comparison vectors

The files in this directory test `scripts/compare_raters.py`.

They are synthetic software-test inputs. They are not empirical scorer submissions and do not establish any finding about Cigna PxDx.

The pair should produce six exact agreements across seven items, one adjacent ordinal difference, no mechanically critical difference, and a passing threshold result.

Participants in the empirical exercise should not inspect these files before submitting their assessments.
