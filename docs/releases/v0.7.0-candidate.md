# Human Influence Telemetry v0.7.0 Candidate Scope

Version `0.7.0` is intended to freeze the first multi-case human replication under the current `0.4.0` assessment contract.

## Release objective

The release will make independent scoring possible without private author explanation. It must publish the full study contract before any scorer begins.

## Required release contents

- locked protocol `HIT-IRP-HIT040-002`;
- machine-readable protocol lock;
- three selected public cases;
- three immutable packet identifiers and versions;
- three decision boundaries;
- three frozen source manifests;
- archived source bundles and SHA-256 digests;
- submission manifest schema;
- scorer instructions;
- coordinator instructions;
- comparison specification and executable code;
- synthetic comparison vectors;
- disagreement taxonomy;
- adjudication template;
- recruitment and eligibility materials;
- validation that blocks scoring when any lock condition remains open.

## Study scale

Three scorers assess three packets. The design produces nine submissions, 24 packet-item units, and 72 pairwise primary comparisons.

## Version layers

- target repository release: `0.7.0`;
- conformance engine: `0.5.0`;
- specification: `0.4.0`;
- assessment schema: `0.4.0`;
- dimension catalog: `0.4.0`;
- application handbook: `0.4.0`;
- protocol: `HIT-IRP-HIT040-002`.

Version `0.7.0` does not revise the normative assessment contract. A contract defect discovered during packet construction requires a separate normative decision before protocol lock.

## Release gate

Tagging is prohibited until:

1. all three packet source surfaces are frozen;
2. every source has a stable identifier or archived digest;
3. the final comparison implementation passes synthetic vectors;
4. scorer-facing and coordinator-only materials are separated;
5. no expected finding appears in scorer-facing files;
6. the protocol lock records `status: locked` and `scoring_permitted: true`;
7. exact-commit validation passes on the release candidate.

## Research boundary

Release `0.7.0` prepares the empirical exercise. It does not itself establish new human agreement evidence, H3 replication, Maturity Level 3, causal effectiveness, legal correctness, or adoption.
