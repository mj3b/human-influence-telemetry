# Human Influence Telemetry v0.4.0 — Candidate Release Notes

## Purpose

Version `0.4.0` stabilizes the normative rubric against the 16 ambiguity classes identified in `HIT-ARFR-001`.

## Candidate contents

- approved evidence-state and finding rules;
- dimension-specific Counsel, Judgment, Command, Correction, Repair, and Reform procedures;
- split Telemetry Integrity model and deterministic derivation;
- actor-authority, sampling, aggregation, contradiction, evidence-reuse, and citation rules;
- candidate assessment schema and dimension catalog;
- 48 executable rubric boundary fixtures;
- explicit migration dispositions for all four public `0.1.0` assessments;
- breaking-change review, migration guide, and adjacent-system claim audit;
- separate released-artifact, candidate-contract, boundary-fixture, and migration validation.

## Research boundary

The release remains Maturity Level 1. It does not establish human inter-rater reliability, field effectiveness, legal compliance, certification, independent adoption, runtime enforcement, signed-receipt interoperability, control harmonization, or evidence portability.

## Human protocol

`HIT-IRP-CIGNA-001` remains locked and unresolved. The earlier planned `v0.3.0` result label is superseded; the eventual result must be published under the next available version without changing the protocol or its pass/fail publication obligation.

## Promotion status

These notes describe the candidate package. The canonical root specification, schema, catalog, handbook, changelog, citation metadata, and release metadata must be updated in a dedicated promotion pull request before `v0.4.0` is tagged.
