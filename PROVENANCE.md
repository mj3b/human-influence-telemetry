## Canonical construct decision

The public instrument uses six substantive dimensions: Counsel, Judgment, Command, Correction, Repair, and Reform.

Telemetry Integrity is a seventh, cross-cutting evidence-quality dimension. It is not merged into the substantive human-influence findings.

## Source-control rule

This public repository is canonical for the technical HIT specification after publication. The private source repository remains canonical for the book chapter and its theological argument.

Changes to the public schema, rubric, or dimension catalog must be versioned here and must not silently inherit unpublished changes from the private source repository.
